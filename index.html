Sobre o Projeto

O OrbitWatch é uma solução em cloud computing hospedada no Microsoft Azure, criada para simular o monitoramento climático via satélite.

A plataforma pode auxiliar:

  • 🌱 Agronegócio
  • 🔥 Prevenção de incêndios
  • 🌧️ Monitoramento climático
  • 🛰️ Sensoriamento remoto

Desenvolvido para a Global Solution - FIAP
